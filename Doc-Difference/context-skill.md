# Context & Skill

- **Context**: Who you are and what you're working with — passive background info the AI reads to understand your environment. Things like your stack, your project structure, your preferences. It doesn't tell the AI to do anything differently, it just informs it.
- **Skill**: How to do a specific thing — an active instruction set that changes the AI's behavior for a particular task. Like "when I ask you to write a React component, always use this pattern, these conventions, this file structure."
