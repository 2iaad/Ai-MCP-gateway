# Project Docs

Drop any project-specific documentation here that you want the AI to reference.

**Examples of what to put here:**
- `api-spec.md` — your API endpoints and contracts
- `data-models.md` — your DB schema or entity relationships
- `architecture.md` — how your services are structured
- `feature-X.md` — context on a specific feature the AI will work on

The more relevant context you give the AI upfront, the less back-and-forth you need.
